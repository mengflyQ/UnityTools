DevXUnity-Unpacker UnityEditor for "Unity 3D 2018.1.0f1" 
Install:
1) Create new empty project on Unity 2018.1.0f1 or higher

2) Set Script runtime version .NET 4.x (restart app)!!!! (on PlayerSettings: Buind setings->Player Settings)

3) Import .unitypackage
4) Open "MainScene"
5) Run
6) Enter licence
7) Restart app, Open "MainScene", Run
